# ERP System   
Dashboard to manage all Products and Orders

## Tech Stack  
**Client:** JavaScript, React
## Run Locally  

Go to the project directory  

~~~bash  
  cd erp-system
~~~

Install dependencies  

~~~bash  
npm install
~~~

Start the server  

~~~bash  
npm run dev
~~~

Copy URL and Open in browser  

~~~  
http://localhost:5173/
~~~

## Features  
- Dashboard    
- Manage Products  
- Manage Orders
